# 📋 RESUMEN DEL PROYECTO BUSSIVEE - SISTEMA PHP + SAP

## 🏗️ ESTRUCTURA DEL PROYECTO
```
/mi_proyecto/
├── /config/
│   └── db.php                 # Configuración base de datos
├── /public/
│   ├── index.php              # Login principal
│   └── registro.php           # Registro de usuarios
├── /src/
│   ├── admin_panel.php        # Panel administración
│   ├── dashboard.php          # Panel usuario normal
│   ├── gestionar_usuarios.php # CRUD usuarios
│   ├── gestionar_productos.php # CRUD productos
│   ├── gestionar_pedidos.php  # Gestión pedidos
│   ├── gestionar_categorias.php # Categorías
│   ├── configuracion.php      # Configuración sistema
│   ├── sap_configuracion.php  # Config SAP
│   ├── sap_web_access.php     # Acceso web SAP
│   ├── sincronizar_sap.php    # Sincronización SAP
│   ├── test_sap.php           # Pruebas SAP
│   ├── theme_manager.php      # Gestor temas
│   ├── reportes.php           # Reportes
│   ├── productos.php          # Catálogo productos
│   ├── mis_pedidos.php        # Pedidos usuario
│   ├── perfil.php             # Perfil usuario
│   └── logout.php             # Cerrar sesión
└── /css/
    └── style.css              # Estilos + temas
```

## 🗄️ BASE DE DATOS

### Tablas Principales:
```sql

config_sap, config_theme, sap_web_sessions
```

### Estructura Usuarios:
- id, nombre, email, password, tipo_usuario, fecha_creacion

### Estructura Productos:
- id, nombre, descripcion, precio, stock, sap_code, creado_por

### Estructura Pedidos:
- id, usuario_id, producto_id, cantidad, total, estado, sap_doc_entry
